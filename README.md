#4press 

4press is a blog engine clone and modified from [blogin](https://github.com/zmmbreeze/blogin). 

###Features


* markdown
* static site
* rss support

###Install
Sorry but cannot install on current stage!


###Config
config.json


###Commands

`$ 4press new myblog`

`$ cd 4press`

`$ 4press post my first post`

`$ 4press update`

`$ 4press server`
